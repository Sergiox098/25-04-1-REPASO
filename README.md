![image](https://github.com/user-attachments/assets/9fe310de-8bf7-46ea-8b45-9830900bcdda)

Sí cumple la condición m = 0, simplemente suma n + 1.
Sí m > 0 y n = 0, Se llama nuevamente la función y a m se le resta 1 y n es igual a 1.
Sí m > 0 y n > 0, Se llama a la función y a m se le resta 1, y para n se llama de nuevo a la función y m se mantiene igual y a n se le resta 1.

Lo más destacable de esta función es que se llama multiples veces, y en una llamada puede realizar otras más, por lo que con números específicos como (6,6) los procesos podrían ser infinitos.
